# Итоговая лабораторная работа. Вариант 4
Данная программа шифрует текст с помощью шифра Цезаря и сохраняет его в формате JSON.
## Инструкция:
### --help - справка о всех командах программы
### --input - ввод текста и его id
### --output - шифрование текстов по введённому id, их вывод на консоль
### --key - генерация ключа
### --exit - выход из программы
