Бенчмаркинг проводился с помощью std::chrono, замерялось время выполнения функции caesar(), которая отвечает за шифровку текста.

	string rand_text(int n) {

		string text;
  
		for (int i = 0; i < n; i++) {
  
			text += char(97 + rand() % 26);
    
		}
  
		return text;
  
	}


string text = rand_text(1000);

auto start = chrono::steady_clock::now();

caesar(text, 10, true);

auto end = chrono::steady_clock::now();

chrono::duration<double> duration = end - start;

cout << "Время выполнения функции составило " << duration.count() << " секунд";

Тем самым, при работе с большими данными программа работает медленнее. Чтобы её оптимизировать, можно использовать другие алгоритмы шифрования текста, допускать меньше копирований.
